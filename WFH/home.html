About this Project

Is it just me, or is it difficult to work from home during the pandemic?

Home is where the heart is, but also where the distractions are. Navigating workday boundaries and expectations to your fam can be tough but neccessary. This project is a fun, mostly tongue-in-cheek, way to communicate my workday boundaries to my housemates. I can update feature flags either through LaunchDarkly's dashboard or by interacting with my LaunchDarkly Slack App.

This website was built with Python, Azure Functions, and LaunchDarkly. See the code here, watch the demo here, or learn how you can deploy your own version here.
